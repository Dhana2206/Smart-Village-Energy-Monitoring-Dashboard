
# Smart Village Energy Monitoring Dashboard

## Project Overview
This system helps village administrators monitor electricity consumption in real-time across different zones (Residential, Commercial, Infrastructure). It features AI-driven energy optimization tips using the Gemini API.

## Grid Features
- **Projected Growth**: Mock data reflects baseline usage typical of infrastructure expansion.
- **Enhanced Visualization**: Real-time load trends with Gemini-powered efficiency scoring.
- **Unified Navigation**: Sidebar provides instant access to Zonal Analytics and Usage Reports.

## Steps to Run
1. **Initialize Project**: Ensure you have Node.js installed.
2. **Setup Gemini API**: Ensure the `API_KEY` environment variable is set.
3. **Login Credentials**:
   - Username: `admin` (or any non-empty string for demo)
   - Password: `admin123` (or any non-empty string for demo)
